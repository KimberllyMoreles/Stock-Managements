# Stock Managements

An AngularJS project made to the first stage of APUS Digital coaching project


## Description

Product management system that provides CRUD operations to 3 entities:

### 1 - Category
- id
- name

### 2 - Product
- id
- category_id
- title
- description
- price
- quantity

### 3 - Stock
- id
- product_id
- quantity
- date_time
